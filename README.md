# Servlet Login Project

A Java Full Stack web application developed using:

- Java
- Servlets
- JDBC
- MySQL
- HTML
- CSS
- Apache Tomcat

## Features
- User Registration
- User Login
- Database Connectivity
- Authentication

## Technologies Used
- Java
- Servlet API
- JDBC
- MySQL
- Eclipse IDE
- Apache Tomcat

## How to Run
1. Import project into Eclipse
2. Configure Apache Tomcat
3. Configure MySQL database
4. Run on server
